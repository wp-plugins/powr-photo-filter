== Installation ==
To install POWr Photo Filter
<br><br>
1. Upload 'powr-photo-filter.php' to the '/wp-content/plugins/' directory
<br>
2. Activate the plugin through the 'Plugins' menu in WordPress
<br><br>
To add POWr Photo Filter as a widget, go to **Appearance->Widgets**, and drag POWr Photo Filter to where you'd like it to appear.
<br><br>
To add Photo Filter within a post or a page, simply write the shortcode **[powr-photo-filter label='MY LABEL']** (including the brackets) where you'd like your Photo Filter to appear. Then visit your page!
<br><br>
Want a more detailed tutorial? Visit [How to add Photo Filter to your Wordpress site](http://www.powr.io/tutorials/how-to-add-photo-filter-plugin-to-your-wordpress-site)

= Will my users see the settings icon in the corner of the Photo Filter? = No! You only see the settings icon because you own the plugins. = I can't see the settings icon anymore, what happened? = You are just logged out of POWr.io. You can show the icon again by typing 'p + up-arrow'.
